Loan Approval Classifier
Machine learning web app that predicts loan approval based on applicant financial data. Supports 4 ML models with real-time predictions and performance visualizations.

Features
4 ML models: Logistic Regression, Random Forest, Gradient Boosting, SVM

Real-time prediction with confidence score

Model accuracy comparison chart

Feature importance analysis
